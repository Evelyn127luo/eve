Programming Languages

Quick description
C C is a general-purpose programming language that gives programmers low-level access to a computer's memory. It's used to create system software, application software, and embedded systems.
C++ C++ is a programming language used to create software. It's an object-oriented language, meaning it uses data fields, or objects, to build programs. C++ is used to develop a wide range of applications, including video games, mobile apps, and operating systems.
Java Java is a programming language used to create applications, websites, and other software. It's a popular choice for developers because it's easy to learn, reliable, and can be used on many different platforms.
C# C# is a popular programming language in game development due to its overall efficiency and compatibility with the Unity game engine. Game developers build the software that makes video games, including the different visuals, sounds, and artificial intelligence (AI) needed for non-character players.
